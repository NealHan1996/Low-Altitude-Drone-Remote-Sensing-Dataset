# Low-Altitude-Drone-Remote-Sensing-Dataset
## Overview

This repository releases **Low-Altitude-Drone-Remote-Sensing-Dataset**, a high-resolution drone image dataset for remote sensing image compression and downstream visual understanding.

Compared with existing UAV datasets, this dataset focuses on:
- high-resolution imagery,
- consistent low-altitude acquisition (~47 m),
- nadir or near-nadir viewpoints,
- residential neighborhoods in coastal urban areas,
- rich structural details including buildings, streets, parking areas, and vehicles.

## Highlights

- High-resolution drone imagery with rich fine-grained structure
- Consistent collection altitude and viewpoint
- Suitable for image compression, restoration, and object detection


## Dataset Structure

- `metadata/`: split files, class definitions, checksums
- `annotations/`: annotation files
- `previews/`: sample images for quick inspection
- `scripts/`: helper scripts for download and verification
